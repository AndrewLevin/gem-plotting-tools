# gem-plotting-tools

1. Setup:

    The following `$SHELL` variables should be defined:

    - $BUILD_HOME
    - $DATA_PATH
    - $ELOG_PATH

    Then execute:

    `source $BUILD_HOME/gem-plotting-tools/setup/paths.sh`

2. Analyzing Scans:
